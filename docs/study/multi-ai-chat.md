# Multi-AI Chat Chrome Extension Study

## 1. Overview

`refs/multi-ai-chat` is a Chrome extension named `Multi-AI Chat`, version `0.1.0`, using Manifest V3 (`refs/multi-ai-chat/public/manifest.json:2`, `refs/multi-ai-chat/public/manifest.json:3`, `refs/multi-ai-chat/public/manifest.json:4`). It exposes a Chrome Side Panel at `sidepanel.html` and uses a module service worker at `background.js` (`refs/multi-ai-chat/public/manifest.json:8`, `refs/multi-ai-chat/public/manifest.json:9`, `refs/multi-ai-chat/public/manifest.json:11`, `refs/multi-ai-chat/public/manifest.json:12`, `refs/multi-ai-chat/public/manifest.json:13`). Its permissions are `sidePanel`, `activeTab`, `tabs`, and `storage`, with HackMD API host permission (`refs/multi-ai-chat/public/manifest.json:6`, `refs/multi-ai-chat/public/manifest.json:7`).

The actual supported provider union is `chatgpt | claude | gemini | grok` (`refs/multi-ai-chat/src/shared/types.ts:3`). The manifest injects content scripts for ChatGPT, Claude, Gemini, and Grok (`refs/multi-ai-chat/public/manifest.json:15`, `refs/multi-ai-chat/public/manifest.json:17`, `refs/multi-ai-chat/public/manifest.json:18`, `refs/multi-ai-chat/public/manifest.json:22`, `refs/multi-ai-chat/public/manifest.json:23`, `refs/multi-ai-chat/public/manifest.json:27`, `refs/multi-ai-chat/public/manifest.json:28`, `refs/multi-ai-chat/public/manifest.json:32`, `refs/multi-ai-chat/public/manifest.json:33`). All content scripts run at `document_idle` (`refs/multi-ai-chat/public/manifest.json:19`, `refs/multi-ai-chat/public/manifest.json:24`, `refs/multi-ai-chat/public/manifest.json:29`, `refs/multi-ai-chat/public/manifest.json:34`).

The build system is Webpack 5. Its entry points are `sidepanel`, `background`, `content/chatgpt`, `content/claude`, `content/gemini`, and `content/grok` (`refs/multi-ai-chat/webpack.config.js:7`, `refs/multi-ai-chat/webpack.config.js:8`, `refs/multi-ai-chat/webpack.config.js:9`, `refs/multi-ai-chat/webpack.config.js:10`, `refs/multi-ai-chat/webpack.config.js:11`, `refs/multi-ai-chat/webpack.config.js:12`, `refs/multi-ai-chat/webpack.config.js:13`). Output is written to `dist` with `[name].js` filenames and `clean: true` (`refs/multi-ai-chat/webpack.config.js:15`, `refs/multi-ai-chat/webpack.config.js:16`, `refs/multi-ai-chat/webpack.config.js:17`, `refs/multi-ai-chat/webpack.config.js:18`). TypeScript is compiled by `ts-loader`, CSS is handled by MiniCssExtractPlugin plus css/postcss loaders, `public` is copied, and HTML is generated from `src/sidepanel/index.html` (`refs/multi-ai-chat/webpack.config.js:20`, `refs/multi-ai-chat/webpack.config.js:23`, `refs/multi-ai-chat/webpack.config.js:24`, `refs/multi-ai-chat/webpack.config.js:28`, `refs/multi-ai-chat/webpack.config.js:29`, `refs/multi-ai-chat/webpack.config.js:41`, `refs/multi-ai-chat/webpack.config.js:42`, `refs/multi-ai-chat/webpack.config.js:43`, `refs/multi-ai-chat/webpack.config.js:46`, `refs/multi-ai-chat/webpack.config.js:47`, `refs/multi-ai-chat/webpack.config.js:48`). The UI stack is React 18, React DOM 18, TypeScript, Tailwind CSS, and PostCSS/autoprefixer (`refs/multi-ai-chat/package.json:32`, `refs/multi-ai-chat/package.json:33`, `refs/multi-ai-chat/package.json:34`, `refs/multi-ai-chat/package.json:11`, `refs/multi-ai-chat/package.json:26`, `refs/multi-ai-chat/postcss.config.js:1`, `refs/multi-ai-chat/postcss.config.js:2`, `refs/multi-ai-chat/postcss.config.js:3`, `refs/multi-ai-chat/postcss.config.js:4`).
